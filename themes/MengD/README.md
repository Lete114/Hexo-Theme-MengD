## 快速开始

安装

稳定[推荐]
``` powershell
git clone -b master https://github.com/lete114/hexo-theme-MengD.git themes/MengD
```

开发
``` powershell
git clone -b dev https://github.com/lete114/hexo-theme-MengD.git themes/MengD
```

打开根目录下的`_config.yml`修改`theme: MengD`
